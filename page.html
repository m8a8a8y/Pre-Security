What happens when you make a DNS request
1.Local Cache Check: Your computer first checks its local cache for the domain name.
2.Recursive DNS Server: If not found locally, a request is sent to your Recursive DNS Server (usually provided by your ISP).checks its cache. If found, it returns the result.
3.Root DNS Server: If not in the cache, the Recursive DNS Server queries a Root DNS Server.
4.TLD DNS Server: The Root DNS Server redirects to the appropriate Top Level Domain (TLD) DNS Server (e.g., .com).
5.Authoritative DNS Server: The TLD DNS Server provides the address of the Authoritative DNS Server for the domain.
6.DNS Record Retrieval: The Recursive DNS Server queries the Authoritative DNS Server and retrieves the DNS record.
7.Response Back to Client: The record is sent back to the Recursive DNS Server, cached, and then relayed to your computer.
